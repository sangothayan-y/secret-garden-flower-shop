# Secret Garden — Flower Shop Website html 

A responsive flower shop landing page built as a **group project**, featuring a hero section, about section with video, product showcase, customer reviews, and contact section.

## Features

- Responsive navigation bar with mobile menu toggle
- Animated hero section
- About section with autoplay video
- Service highlights (delivery, returns, offers, secure payment)
- Product showcase grid
- Customer review section
- Contact section

## Tech Stack

- HTML5
- CSS3
- [Font Awesome](https://fontawesome.com/) (icons, via CDN)

## How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/sangothayan-y/secret-garden-flower-shop.git
   cd secret-garden-flower-shop
   ```
2. Open `index.html` in your browser. No build steps or installs needed.

## Project Status

This was built as a group academic project for practicing front-end web development fundamentals (HTML/CSS layout, responsive design, and basic UI structure). Content (placeholder text, product info) is for demonstration purposes only.
